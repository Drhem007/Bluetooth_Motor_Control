Bluetooth-Bridge Motor Control System

An IoT ecosystem bridging a modern **Flutter** mobile interface with **Arduino** embedded hardware. This project abstracts low-level PWM signals into user-friendly operational modes.

* **Intuitive HMI**: Responsive Flutter app for real-time device discovery and control.
